# public/

Drop `banner.jpg` here (or `banner.png` / `banner.webp`, then adjust the
`src` in `components/Banner.tsx`). The dashboard will serve it directly
at `/banner.jpg`. If the file is missing, the banner gracefully falls
back to the inline SVG `QuaiScene`.

Recommended: 1800×600 (3:1), JPEG ≤ 300 KB.
