# Glaces en Seine · Mission Control

Dark-mode operations dashboard for a food caravan on the Quai de Seine
(95240 / 95530). Reads weather + event signals and tells the operator how much
batter to prep, what topping to feature, when the Sunday pulses are coming,
and whether Grand-Mère is about to inspect the counter.

```bash
npm install
npm run dev
```

See [`CLAUDE.md`](./CLAUDE.md) for architecture, the formulas, the API
feedback (what's live vs. stubbed), and the roadmap.
